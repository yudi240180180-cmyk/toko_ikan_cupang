# 🐟 Web Toko Ikan Cupang (Laravel)

Aplikasi E-Commerce sederhana berbasis framework Laravel untuk manajemen stok dan penjualan berbagai jenis ikan cupang (Plakat, Halfmoon, Giant, dll).

---

## 🛠️ Fitur Utama
- **Katalog Produk:** Daftar ikan cupang beserta detail dan harga.
- **Manajemen Stok:** Update data ikan melalui panel admin (CRUD).
- **Responsive Design:** Tampilan web yang rapi di berbagai perangkat.

## 📦 Prasyarat Instalasi
- PHP >= 8.1
- Composer
- MySQL (XAMPP)
